破解VBA Project密码

2019-11-12

使用二进制编辑器,使用查找功能定位到DPB=”…….上

把DPB更改成DPX后保存

重新打开文档会提示错误未找到关键字DPx是否继续加载工程,选择 是

Alt+F11打开VBA工程，右键，VBAProject属性,保护,重新输入一个新密码,保存文档

再次打开就可以用新密码解锁VBA工程
