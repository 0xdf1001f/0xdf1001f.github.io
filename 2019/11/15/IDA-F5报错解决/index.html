IDA F5报错解决

2019-11-15

write access to constant memory detected. output maybe wrong.

在IDA中，有时候F5会出现

write access to constant memory detected. output maybe wrong.

在对应的报错代码段，按下Shift+F7，打开Segment视图，修改对应的区段属性

添加“W”属性，修改Segment Class为“DATA”

修改后在F5即可解决

Positive sp value has been found

有时候F5，会提示

Positive sp value has been found

这个问题是栈帧不平衡导致的，手动修改栈帧

先到达报错地址附近

可以发现在push了4个参数之后，使用add esp 40h来平衡，一个参数0X4，四个参数0x10

选中add esp,40h，按下Alt+K，修改为0x10

修改后，再F5

有时候有其他情况，没有add esp平衡堆栈

只需要到报错地址的前面，修改前面的堆栈即可
